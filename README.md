#  MediTrack - Smart Access to Medicines

MediTrack is a full-stack web application that helps users search, reserve, and locate medicines from nearby pharmacies. It provides an easy-to-use platform for customers, pharmacy owners, and administrators.

##  Features

-  Search medicines
-  Reserve medicines
-  Pharmacy inventory management
-  Customer, Pharmacy & Admin Login
-  Pharmacy location support
-  Inventory management
-  Secure authentication

##  Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
